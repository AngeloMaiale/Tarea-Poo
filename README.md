# Tarea-Poo

Resumen: Relicario de Hielo: El legado del Norte es una historia que se juega en la consola. El que juega es Eira Valen. Ella busca tomar la Reliquia del Norte. Esto es para arreglar el honor de su familia. El juego mezcla buscar sitios, pelear por turnos, resolver enigmas, tomar decisiones difíciles y manejar cosas. Todo pasa en una cueva muy fría con tres niveles.

Cómo se juega:

Inicia el juego usando Main. java.

Avanzarás por cuartos en cada piso. Encontrarás bichos, trampas, retos y que decidir.

Antes de cada cuarto, vas a un sitio para prepararte. Allí puedes:

Descansar una vez por cuarto. Esto arregla tu vida (HP) y energía (EN).

Usar cosas de tu caja de objetos.

Poner trucos (cheats).

Pasar el cuarto si el truco correcto está activo.

Cosas clave:

Pelea:

El modo es por turnos. Los enemigos pegan al azar. A veces fallas tus golpes.

Puedes pegar, usar fuerzas (gastan EN), usar ítems o tratar de correr.

Algunos bichos tienen modos especiales. Como sanar o ponerse bravos.

Retos y trampas:

Hay cuartos con puzzles. Resuélvelos con sentido común o buscando pistas.

Puedes quitar trampas si eres bueno en eso. Si fallas, recibes daño.

Caja de objetos e ítems:

Encontrarás cosas como pociones, vendas, luces, pilas de fuerza, y más.

Cada cosa sirve para algo distinto: curar, dar energía, subir el daño por poco tiempo, etc.

Puntos y niveles:

Ganas puntos al ganar peleas, arreglar enigmas, evitar trampas o al pasar cosas grandes.

Cuando subes de nivel:

Tu vida y energía máximas crecen.

Recuperas salud.

Cada tres niveles, tu golpe fuerte mejora para siempre.

Trucos listos: Puedes activar estos desde el sitio de preparación:

Vida sin fin: No recibes daño alguno.

Golpe único: Matas bichos al instante.

Pasar cuartos: Te deja omitir cuartos enteros.

Ojo: Si usas un truco, el final dirá que todo fue mentira. Esto avisa para que se cuide el reto serio.

Finales que se pueden ver:

Final A: Ganas de verdad. Tomas la Reliquia sin hacer trato con el que traicionó.

Final B: Ganas con trato. Tomas la Reliquia, pero hiciste un pacto con el traidor.

Final C: Pierdes. Mueres, dejas la Reliquia o no logras salir bien.

Adicionales:

Puedes empezar de nuevo o cerrar el juego al final.

Cosas como Cuero o Fichita 3 sirven para abrir cosas nuevas o arreglar retos.

El juego se hizo para crecer. Es fácil meter más cuartos, bichos, cosas o sucesos.

Se necesita:

Java 8 o mejor versión.

Iniciar desde Main. java.
